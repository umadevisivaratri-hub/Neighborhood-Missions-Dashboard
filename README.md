# Neighborhood-Missions-Dashboard
Full-stack assignment project demonstrating REST API development, database integration, and scalable application design using modern technologies.
## Overview
A full-stack application built as part of an assignment to demonstrate REST API integration, frontend development, and scalable architecture.

## Tech Stack
- React + TypeScript
- Vite
- REST APIs

## Features
- Data visualization dashboard
- API integration
- Responsive UI
